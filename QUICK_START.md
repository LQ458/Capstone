# 快速开始指南

## 🚀 5分钟快速体验

### 1. 运行游戏
```bash
python3 tic_tac_toe.py
```

### 2. 选择游戏模式
- 输入 `1` 进行人机对战
- 输入 `2` 观看AI对战

### 3. 开始游戏
- 输入坐标格式：`行 列`（如：`1 1`）
- 行和列的范围都是 0-2

## 📊 查看实验结果

### 运行数据分析
```bash
python3 simple_visualization.py
```

### 生成图表
```bash
# 激活虚拟环境
source venv/bin/activate

# 生成图表
python3 create_charts.py
```

## 📄 编译报告

### 自动编译
```bash
./compile_report.sh
```

### 手动编译
```bash
pdflatex report.tex
```
